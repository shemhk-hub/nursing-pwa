✅ WHAT EACH FILE DOES
═══════════════════════════════════════════════════════════════════════════════

📂 src/lib/
├── supabase.ts
│   └─ Database client for Supabase
│   └─ Functions to fetch courses, users, bookmarks, ratings
│   └─ Search and subscription management
│
├── auth-context.tsx
│   └─ React Context for authentication
│   └─ Provides user state across app
│   └─ useAuth() hook for accessing auth info
│
└── theme-provider.tsx
    └─ React Context for dark mode
    └─ localStorage persistence
    └─ useTheme() hook for toggling theme

📂 src/app/
├── layout.tsx
│   └─ Root layout component
│   └─ Wraps app with AuthProvider and ThemeProvider
│   └─ Includes metadata for SEO
│
├── page.tsx (HOME PAGE)
│   └─ Landing page at /
│   └─ Shows login/signup buttons for guests
│   └─ Shows "Go to App" for logged-in users
│   └─ Beautiful welcome message
│
├── globals.css
│   └─ Global styles for entire app
│   └─ Dark mode support
│   └─ Reset and base styles
│
├── api/auth/route.ts
│   └─ API endpoint for OTP operations
│   └─ Handles: send OTP, verify OTP
│   └─ Returns success/error responses
│
└── app/page.tsx (STUDENT DASHBOARD)
    └─ Student home page at /app/home
    └─ Shows welcome message
    └─ Displays quick action cards
    └─ Requires authentication (redirects to login if not)

💡 WHAT'S HAPPENING UNDER THE HOOD
═══════════════════════════════════════════════════════════════════════════════

1. USER VISITS YOUR APP
   ↓
2. LAYOUT.TSX LOADS
   ├─ Wraps with AuthProvider (checks if user is logged in)
   ├─ Wraps with ThemeProvider (dark mode support)
   └─ Loads globals.css
   ↓
3. HOME PAGE (page.tsx) DISPLAYS
   ├─ Shows login/signup if not logged in
   └─ Shows app link if logged in
   ↓
4. USER LOGS IN
   ├─ Auth context updates
   └─ User can now access /app/home
   ↓
5. STUDENT DASHBOARD (/app/home) LOADS
   ├─ Checks if user is logged in
   ├─ If not → redirects to /auth/login
   └─ If yes → shows dashboard
   ↓
6. DATABASE READY
   ├─ Supabase client connected
   ├─ Can fetch courses, users, content
   └─ All 13 tables ready to use
